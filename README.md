# Vue.js 購物網站-ADMIN管理系統

## Demo
https://syj0905.github.io/PlayStation-admin/dist
# 功能
本作品其主要功能有：
* 後台:商品管理、客戶訂單管理、優惠券管理
# 練習技術
* Vue Cli 2.0
* Vue Router
* Vue Component
* API的應用
* Bootstrap 4 + 手刻 SCSS 的混合應用
